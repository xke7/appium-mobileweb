Appium Python - Mobile Web Explore
---
# Install
Android environment set up
```shell
export ANDROID_HOME=$your_local_sdk
```

install appium
```shell
npm install -g appium
```

start appium
```shell
appium
```

install python libraries
```shell
pip install Appium-Python-Client
pip install pytest
```

# Usage
```shell
py.test $filename
```
